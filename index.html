Welcome to blast2galaxy

blast2galaxy is a Python API and CLI to perform BLAST queries against Galaxy servers that have the NCBI BLAST+ tools [1] installed.

Figure 1: Blast2galaxy provides a high-level convenience layer between any Galaxy server with NCBI BLAST+ tools installed and different types of clients and usage scenarios. Researchers, applications and computational pipelines can either use the Python-API or the CLI of blast2galaxy to send requests for a BLAST search to any compatible Galaxy server. The use of BLAST+ tools by multiple applications and the provision of corresponding BLAST databases can be centralized and made reusable by use of a Galaxy server.

Please read the CLI Reference if you want to use blast2galaxy on the command line as drop-in replacement for NCBI BLAST+ tools or read the API Reference if you want to use blast2galaxy inside a Python application to perform BLAST requests against a Galaxy server.

References

[1] Peter J. A. Cock, John M. Chilton, Björn Grüning, James E. Johnson, Nicola Soranzo, NCBI BLAST+ integrated into Galaxy, GigaScience, Volume 4, Issue 1, December 2015, s13742-015-0080-7, https://doi.org/10.1186/s13742-015-0080-7
